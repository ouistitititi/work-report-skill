## 快速开始

### 首次使用

首次调用 Skill 时，会引导你完成初始化配置：

```
步骤1：存储位置
→ 报告和记录存放在哪个文件夹？默认 ./工作记录/

步骤2：报告格式
→ 1. Markdown（推荐） 2. Word 3. PPT

步骤3：详细程度
→ 1. 极简版 2. 标准版 3. 深度版

步骤4：固定模板
→ 是否有公司固定模板？
```

## 记录工作

### 基本用法

直接告诉 Skill 你做了什么：

```
完成需求文档初稿
参加部门周会
修复登录页面Bug
```

### 带项目标记

使用 `[项目名]` 或 `#项目名` 标记项目：

```
[项目A] 完成需求文档初稿
修复登录Bug #项目B
项目C：数据库设计评审
```

### 带状态和备注

```
[项目A] 完成需求文档初稿，进度50%，需产品确认
修复登录Bug #项目B，已提交代码
```

## 多项目管理

### 项目标记方式

| 方式 | 示例 |
|------|------|
| 方括号 | `[项目A] 完成需求文档` |
| 井号 | `完成需求文档 #项目A` |
| 冒号 | `项目A：完成需求文档` |

### 多项目报告生成

检测到多个项目时，会询问：

```
检测到多个项目（项目A、项目B、项目C），如何生成？
A. 合并生成一份报告
B. 按项目拆分生成多份报告
```

## 可视化建议

### 触发条件

报告内容包含以下数据时自动提示：
- 进度数据：`进度75%`、`完成度80%`
- 耗时数据：`耗时2h`、`花费3天`
- 对比数据：`环比增长20%`

### 图表类型

| 图表 | 适用场景 |
|------|----------|
| 甘特图 | 进度、时间线 |
| 饼图 | 占比分布 |
| 柱状图 | 对比数据 |

## 常见问题

### Q: 如何查看历史记录？

```
查看本周记录
查看4月的记录
```

### Q: 记录存放在哪里？

默认存放在 `./工作记录/` 目录下：
- `records/工作记录_YYYY-MM.md` - 工作记录
- `reports/周报-YYYY-WXX.md` - 生成的报告

### Q: 如何导出报告？

```
导出周报到桌面
导出月报为Word格式
```

### Q: 支持哪些报告格式？

- Markdown（默认，推荐）
- Word
- PPT
