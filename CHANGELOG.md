# Changelog

All notable changes to this project will be documented in this file.

## [2.0.0] - 2026-04-24

### Added
- 🗂️ **多项目管理** - 支持项目标记（`[项目名]` 或 `#项目名`），自动归类
- 🔧 **智能纠错** - 语义回溯，支持修改/删除/补充历史记录
- 📊 **可视化建议** - 检测进度、耗时数据时自动建议生成图表
- ⚡ **极简沟通** - 拒绝废话，Token节省原则

### Changed
- 存储格式从 JSON 改为 Markdown 表格，更易读易编辑
- 优化报告生成逻辑，优先按项目整合，其次按类别整合
- 简化初始化流程，减少冗余询问

### Fixed
- 修复重复内容未合并的问题
- 修复多项目报告生成时的排序逻辑

## [1.0.0] - 2026-04-24

### Added
- 📝 对话式工作记录
- 📊 周报/日报/月报生成
- ⚙️ 初始化配置（存储位置、格式、详细程度）
- 📄 自定义模板支持
